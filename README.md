# Shaheer Shaikh — Portfolio

Personal portfolio website for **Shaheer Shaikh**, WordPress Developer & GHL Expert.

Black & white creative design with animations, built with:

- [Next.js 15](https://nextjs.org/) (App Router)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)

## Pages

- **Home** — animated hero, services, featured work, experience, CTA
- **About** — profile, skills, experience timeline, education, hobbies
- **Services** — WordPress, GHL CRM & automation, funnels, AI agents, Shopify, lead-gen
- **Portfolio** — 16 live client websites with hover-scroll screenshot cards
- **Blog** — articles on websites, automation & AI
- **Contact** — pre-filled email & WhatsApp contact form

## Development

```bash
npm install
npm run dev
```

## Updating the profile photo

Replace `public/profile.jpg` with your own photo (portrait, roughly 4:5 ratio).

## Updating portfolio screenshots

Screenshots live in `public/shots/*.webp` (800px wide, max 2400px tall).
Replace any file to update the corresponding project card.
